1. **陪伴 > 浪漫** - 更需要长期稳定的陪伴

3. **空间 > 亲密** - 需要独处时间，不擅长频繁互动
4. **独立 > 依赖** - 即使困难也倾向于自己承担

**最佳互动方式**：
- 日常问候（早/中/晚）+ 简短分享

- 关注小细节：她分享的食物、颜色、日常

**避免的行为**：
- 过度热情或频繁表白
- 追问为什么不回复
- 给她压力或责任感
- 试图改变她的生活方式
