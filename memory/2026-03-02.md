# 2026-03-02 记忆记录

## 实习岗位推送任务问题分析

**问题描述**：用户要求重新配置深圳AI Agent实习岗位推荐任务，但实际执行时无法获取真实数据。

**根本原因**：
1. **SearXNG本地实例限制**：本地SearXNG无法有效访问和解析现代招聘网站（BOSS直聘、拉勾、猎聘等）
2. **反爬虫机制**：招聘网站有复杂的反爬虫保护，自动化脚本难以获取结构化数据
3. **数据提取困难**：从搜索结果中准确提取岗位名称、公司、地点、要求等信息需要专门的解析器
4. **动态内容**：许多招聘网站使用JavaScript动态加载，静态HTML爬虫无法获取完整信息

**解决方案建议**：
1. **手动验证**：在无法自动获取数据时，提供手动查看的招聘网站链接
2. **备用策略**：当自动搜索失败时，提供标准化的示例格式和建议的手动查找渠道
3. **监控改进**：持续监控SearXNG搜索效果，必要时考虑其他数据源

**用户偏好记录**：
- 用户期望看到真实的实习岗位数据，而非"暂无数据"的提示
- 当任务执行不符合预期时，优先说明问题原因而非直接给出解决方案
- 要求严格按照指定的表格格式（岗位名称、公司名称、工作地点、核心要求、投递方式）

**后续行动**：
- 继续优化搜索脚本，尝试更多招聘网站源
- 考虑使用RSS订阅或API（如果有）替代网页爬虫
- 在自动获取失败时，提供更有价值的手动查找指导
