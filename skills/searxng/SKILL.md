# SearXNG Privacy-Respecting Metasearch

This skill enables privacy-respecting web search using your local SearXNG instance.

## Usage
- Use when you need to search the web without external API dependencies
- Supports web, images, news, and other search types
- Respects user privacy by using local search instance

## Local Instance
- URL: http://localhost:8080
- Docker container should be running
- No API keys required
